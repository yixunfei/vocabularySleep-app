## 二、重构尝试总结

### 尝试的方法

**委托模式** - 在 AppState 中使用 WordbookState:

```dart
class AppState {
  late final WordbookState _wordbookState;
  
  // 委托 getters
  List<Wordbook> get wordbooks => _wordbookState.wordbooks;
  Wordbook? get selectedWordbook => _wordbookState.selectedWordbook;
  // ...
}
```

### 遇到的问题

1. **SearchMode 类型冲突**
   - AppState 和 WordbookState 都定义了 SearchMode
   - 需要统一导入

2. **大量 Getters 需要委托**
   - 约 40 个 getters 需要更新
   - 涉及多个领域 (词本、播放、练习等)

3. **测试文件需要同步更新**
   - 多个测试文件引用 AppState
   - 需要更新 mock 和 stub

4. **技术复杂度**
   - 3011 行大文件重构风险高
   - 需要更详细的规划和测试

### 方案 A: 渐进式重构 (推荐)

**阶段 1**: 创建新模块，保持旧代码 (2 天)
- 创建 WordbookState

- 添加完整测试

**阶段 2**: 逐步迁移 (3 天)
- 每次迁移 1-2 个方法

**阶段 3**: 删除旧代码 (1 天)
- 确认所有功能正常
