## 版本历史

### [未发布] - 2026-04-09

### 文档 (Docs)
- 新增单词本结构标准化评估计划，基于当前 `wordbooks/` 示例词本、导入逻辑、SQL 存储和展示链路，整理统一格式建议、迁移方向和风险矩阵。
- 新增 `wordbook.v1` 单词本标准文档、标准模板和最小示例文件，作为后续批量标准化处理的统一依据。
- 新增 `PLAN_002_toolbox_ui谨慎重建与结构优化.md`，约束本轮工具箱 UI 重建仅涉及展示层、结构组织和规范优化，不改动业务逻辑。

### 修改 (Changed)
- 重建工具箱入口页的展示层结构：将入口数据、页面组件和样式常量拆分为独立文件，降低 `toolbox_page.dart` 的耦合与维护成本。
- 工具箱入口卡片调整为更克制的圆角、渐变层次、阴影与按压反馈，并统一移动端内容最大宽度为 600dp。
- 工具箱通用页面壳 `ToolboxToolPage` 与 `ToolboxMetricCard` 同步接入共享尺寸常量，统一内容宽度和基础卡片样式。
- 对 `疗愈音钵` 页面进行第二轮 UI 精修：提升频率选择器触控宽度与选中反馈，优化移动端抽屉中的频率条层次和氛围动画节奏。
- 对 `呼吸训练` 页面进行第二轮 UI 精修：重做场景选择 chip 的视觉表达，并抽取统一的呼吸信息卡样式组件，收敛概览卡、摘要卡和设置卡的展示结构。
- 对 `舒缓音乐` 页面进行第三轮视觉精修：增强模式筛选、轨道 pill 与控制按钮的层次表达，并让 Blob 氛围层更不对称、更有体积感，降低机械感。
- 对 `静心念珠` 页面进行第三轮视觉精修：统一 Hero 卡与设置卡的柔和面板语言，重做圈数规格 chip 与材质卡的选中态层次。
- 将本地分析验证脚本扩展为“本地验证工具箱”：统一封装格式检查、格式化写回、`dart analyze`、`flutter analyze`，并固定将 Dart/Flutter 的临时配置目录重定向到工作区内。
- 本地验证工具箱新增对 PowerShell 目标的语法校验分流，避免对 `.ps1` 文件误用 `dart format`，便于脚本自身也纳入本地校验流程。
- 继续收敛 `toolbox` 共享展示层：新增通用面板、选择 pill、信息 pill 与图标按钮组件，并让入口页、呼吸训练、疗愈音钵、静心念珠接入统一的卡片与交互外壳。
- 将若干页面内部的重复样式表达下沉到 `lib/src/ui/pages/toolbox/`，减少 Frosted 面板、胶囊按钮和标签 pill 的散落实现，提升后续维护与扩展一致性。
- 新增固定 `MiniMax-M2.7` 调用模板脚本与多模型 fan-out 调度脚本，将 `opencode` 外部模型协作能力固化为仓库级开发工具。
- 新增 `opencode` 模型 profile 配置文件，集中维护常用模型别名和实际 provider/model 映射，降低后续切换成本。
- 对照 `docs/toolbox_design` 重新补齐剩余设计反馈：扩展 `app_motion.dart` 的缓动与时长层级，新增 `toolbox_colors.dart` 并将工具箱入口 accent 收敛为统一色彩令牌。
- 继续完成页面级反馈闭环：将 `疗愈音钵` 的敲击反馈收敛到 800ms/8s 的目标范围，给 `禅意沙盘` 增加双指误触缓冲与轻量提示，并让 `舒缓音乐` 的 Blob 背景基于现有轨道控制器产生更自然的漂移与尺度变化。
- 对 `D:\workspace\wordbooks_cover\result` 下 31 份标准化单词本执行 `wordbook.v1` 结构验收与抽样复核，确认顶层结构、核心字段、`entry_count`、`entry_id` 唯一性和结构化数组形态均满足当前标准，可作为后续系统重构的文件基线。

### 风险变更 (Risk)
- 本轮仅重构 `toolbox` 展示层与结构组织，保留原有入口分组、文案语义和 `pageBuilder` 跳转逻辑，避免引入功能行为变更风险。
- 第二轮精修继续限定在 `疗愈音钵` 与 `呼吸训练` 的 UI 层，不调整音频播放、计时阶段或场景切换逻辑。
- 第三轮精修继续限制在 `舒缓音乐` 与 `静心念珠` 的视觉层，不修改播放逻辑、轨道切换、念珠物理反馈或计数逻辑。
- 本轮共享样式收敛继续限制在展示容器、视觉反馈与文件组织层，未调整任何业务判断、持久化字段、状态来源或交互逻辑。
- 外部模型脚本仅作为开发辅助工具接入，默认不直接回写业务代码，避免把不完整上下文下生成的建议误当成最终实现。
- 本轮设计反馈闭环修复优先采用参数收敛、令牌抽取和轻量交互缓冲，不重写既有业务链路，以降低对绘制、播放和手势主流程的回归风险。
- 本次词本验收结论仅代表“结构标准通过”，不等同于所有词条的语义质量、译文自然度和词性标注都已完成业务级验收，后续仍需安排内容质量抽检。
