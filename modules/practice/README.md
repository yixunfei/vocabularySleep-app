# practice 模块说明

## 模块概述
`practice` 模块承载练习会话、错题追踪、统计汇总与练习回顾展示。

## 架构设计（当前阶段）
- 当前仍由 `AppState` 承载主要练习状态与行为编排。
- 已完成模块开关接入：可通过模块管理页控制入口可见性与访问性。
- 已完成仓库首刀接入：练习域数据库读写通过 `PracticeRepository` 进入数据层。
- 后续阶段将继续拆分为独立状态容器与用例层。

## 使用指南
- 入口来自底部导航 `practice` tab。
- 依赖词本可见范围与练习设置，支持错题本联动。

## 注意事项
- 本模块是 `PLAN_024` 阶段 2 的优先试点域。
- 拆分时需优先保证判题、会话切题性能与统计一致性不回退。
