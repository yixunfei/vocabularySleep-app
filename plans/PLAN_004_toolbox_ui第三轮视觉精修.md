# 计划 004: toolbox UI 第三轮视觉精修

## 基本信息
- **创建日期**: 2026-04-09
- **状态**: 进行中
- **负责人**: Codex

## 目标
继续遵循“只动 UI、不动逻辑”的边界，对 `舒缓音乐` 与 `静心念珠` 两个工具页做第三轮视觉精修，优先处理设计反馈中仍未覆盖的氛围层、卡片层次与选中态表达问题。

## 范围边界
- **包含**：
  - `toolbox_soothing_music_v2_page.dart` 中 Blob 氛围层、筛选 chip、轨道 pill 和控制按钮的视觉优化
  - `toolbox_prayer_beads_tool.dart` 中 Hero 卡、设置卡、圈数规格 chip 和材质卡的视觉统一
  - 与上述 UI 精修直接相关的轻量样式辅助代码
- **不包含**：
  - 音频播放、轨道切换、排列逻辑、全屏逻辑
  - 念珠拨动、物理动画、声音触发、计数和持久化逻辑

## 详细步骤
1. 梳理舒缓音乐与静心念珠在设计反馈中的剩余问题，筛选仅属于视觉层的调整项。
2. 精修舒缓音乐页面的氛围层和交互控件：
   - 降低 Blob 的机械感，增强体积感和不对称层次
   - 统一模式筛选 chip、轨道 pill 和控制按钮的选中与按压反馈
3. 精修静心念珠页面：
   - 提升 Hero 卡与设置卡的层次感
   - 重做圈数规格与材质卡的选中态和移动端触控表达
4. 更新 `CHANGELOG` 记录本轮 UI 层精修，不引入功能行为变更。

## 风险评估
- **风险 1**: 舒缓音乐氛围层样式变化影响文字对比度
- **缓解措施**: 保留原有调色板与文本色来源，只增强边缘层次和模糊体积

- **风险 2**: 念珠设置区卡片重绘后削弱原有材质辨识
- **缓解措施**: 不改变材质调色逻辑，仅增强边框、渐变和选中反馈
